# 家族ニュース

家族・親戚でSNS作りをしたい家族向けの家族ニュースというプロダクトはSNSです。  
これは、フォローが不要で、非常に簡単に導入でき、LINEとは違って、等親数を指定するだけで必要なグループにメッセージを送ることができます。

## トレードオフスライダー

２、プロダクトの品質  
３、アピールしたい機能を含んでいること  
４、スコープ  
５、納期
